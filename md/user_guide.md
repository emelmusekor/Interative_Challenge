# Edu_Bridge AI 영재 교육 프로그램 - 사용자 가이드

이 문서는 5대 지능 영역의 25개 과제에 대한 사용 방법과 학습 목표를 설명합니다.

## 1. Vision AI (시각 지능)
| 과제 | 사용 방법 |
| :--- | :--- |
| **A1. 숨은 정보 찾기** | 화면의 복잡한 배경 속에서 제시된 사물을 찾아 클릭하세요. |
| **A2. 방해물 지우기** | 목표 물체를 가리고 있는 방해물(구름, 먼지 등)을 클릭하거나 문질러 제거하세요. |
| **A3. 미로 탈출** | 시작점에서 출발하여 벽에 닿지 않고 목표 지점까지 이동하는 경로를 그리거나 키보드로 이동하세요. |
| **A4. 패턴 찾기** | 제시된 패턴과 동일한 무늬나 규칙을 가진 카드를 찾아 짝을 맞추세요. (드래그앤드롭 권장) |
| **A5. 베스트 포토그래퍼** | 마우스를 움직여 별이 가장 많이 보이는 위치를 찾고, 클릭하여 카메라를 고정한 뒤 촬영 버튼을 누르세요. |

## 2. Logic AI (논리 지능)
| 과제 | 사용 방법 |
| :--- | :--- |
| **B1. 명령어 읽기** | 아이콘(명령어)의 순서를 보고 캐릭터가 이동할 최종 위치를 예측하여 선택하세요. |
| **B2. 문제 쪼개기** | 큰 프로젝트(루트 노드)를 완료하기 위해 하위 단계(자식 노드)들을 순서대로 체크하여 완료하세요. |
| **B3. 흐름 만들기** | 조건(If)에 따라 길이 갈라집니다. 조건에 맞는 올바른 길을 선택하여 흐름을 완성하세요. |
| **B4. 오류 고치기** | 뒤죽박죽 섞인 명령어 블록을 올바른 순서대로 드래그하여 재배열하세요. |
| **B5. 청소의 달인** | 청소 구역(상자)을 클릭하여 로봇들에게 골고루 나누어 주세요. (클릭 시 로봇 순차 배정) |

## 3. Data AI (데이터 지능)
| 과제 | 사용 방법 |
| :--- | :--- |
| **C1. 재료 모으기** | 화면에 떨어지는 다양한 아이템 중 미션에 필요한 재료만 바구니로 받으세요. |
| **C2. 스마트 분류기** | 컨베이어 벨트 위의 게이트를 클릭하여 방향을 바꾸고, 물건을 올바른 색상의 상자로 보내세요. |
| **C3. 암호 해독** | 암호화된 문자를 보고 규칙(밀기, 뒤집기 등)을 파악하여 원래 문장으로 해독하세요. |
| **C4. 정답 확인** | 데이터 목록에서 규칙에 맞지 않는 데이터(오류)를 찾아내세요. |
| **C5. 짐 싸기** | 제한된 공간(가방)에 물건을 효율적으로 채워 넣으세요. (테트리스 형태) |

## 4. Network AI (관계 지능)
| 과제 | 사용 방법 |
| :--- | :--- |
| **D1. 마니또 찾기** | 화살표 방향을 보고 누가 누구에게 영향을 주는지 파악하여 질문에 답하세요. |
| **D2. 가지치기** | 복잡한 네트워크에서 불필요한 연결(약한 고리)을 끊어 핵심 그룹을 남기세요. |
| **D3. 관계 잇기** | 흩어진 점(노드)들을 규칙에 맞게 선으로 연결하여 네트워크를 완성하세요. |
| **D4. 모두 연결하기** | 모든 집(노드)이 하나도 빠짐없이 연결되도록 전선을 최소한으로 설치하세요. |
| **D5. 장난감 교환** | 노드를 드래그하여 보기 좋게 정리하고, 시작부터 끝까지 최소 비용 경로를 순서대로 클릭하세요. |

## 5. System AI (시스템 지능)
| 과제 | 사용 방법 |
| :--- | :--- |
| **E1. 상황 파악하기** | 그림이나 소리를 단서로 현재 상황(비 오는 날, 공사장 등)을 맞추세요. |
| **E2. 역할 놀이** | 주어진 상황(병원, 학교 등)에 꼭 필요한 역할(의사, 선생님 등)을 드래그하여 빈칸에 채우세요. |
| **E3. 이벤트 마스터** | 특정 조건(예: 빨간 원)이 등장했을 때만 빠르게 반응(클릭)하세요. |
| **E4. 데이터 탐험가** | 조건(예: 가격 500원 이하)에 맞는 데이터를 목록에서 찾아 체크하세요. |
| **E5. 성능 높이기** | 한정된 자원을 여러 프로세스에 할당하여 전체 처리량을 최대화하세요. |

## 상태 점검 및 개선 계획
- **현재 상태**: 대부분의 과제가 클릭 기반으로 동작.
- **개선 필요**: A4(패턴), B1(명령어), B4(오류 수정), C1(수집), C5(짐싸기) 등에 드래그앤드롭 도입 필요.
